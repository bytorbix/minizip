# minizip

A simple file compression tool written in C using Run-Length Encoding (RLE).

## Build

```bash
make
```

## Usage

```bash
./minizip compress <source> <destination>
```

**Example:**
```bash
./minizip compress input.txt output.tsur

./minizip decompress input.txt output.tsur
```

## make

```bash
make 
```

```bash
make clean
```
